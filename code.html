Code explanation below.

getValues()

The code is broken into separate functions to divide the necessary steps before bringing them back together to display the result onto the page.

The getValues( ) function starts and ends the entire process. Starting by identifying what the user-entered start and end values are, converting them to integers and then validating the data before passing it on to the next step.

generateNumbers()

The generateNumbers( ) function creates an array with every number integer from the start value to the end value (inclusive) and then passes it to the final step.

displayNumbers()

The displayNumbers( ) function iterates through the array that is passed in. On each iteration it assigns a "class" variable either an even or odd value depending on if the current value in the array that is being accessed is even or odd.

Then it injects the currently accessed number and its corresponding class into the table to be displayed. From here we access the injected class in CSS to change the font-weight to bold for every even value.
